# Dashboard component

Simple static dashboard scaffold demonstrating:

- Header with welcome message and live date/time
- Membership cards (active plan + expiry countdown)
- Quick summary stats
- Quick action buttons with hover states

To view:

1. Open `dashboard/index.html` in a browser (double-click or run `open dashboard/index.html` on macOS).
2. Edit `script.js` to change `USERNAME` and `EXPIRY_ISO` values.
